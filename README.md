# Covid-19 Cases Tracker Application 

### An application to track the cases of covid-19 in the world, grouped by country and state
Technologies used:
- FrontEnd - ThymeLeaf
- BackEnd - Java Spring Boot

---

Source of data: https://raw.githubusercontent.com/CSSEGISandData/COVID-19/master/csse_covid_19_data/csse_covid_19_time_series/time_series_covid19_confirmed_global.csv

---

Deployed using heroku: https://worldcovid19casestracker.herokuapp.com
